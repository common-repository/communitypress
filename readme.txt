== Description ==

A social community in a box! Based off Buddypress, With a many additional features like a member livesearch, friend tagging in photos, media & wall posts, facebook integration, user privacy, improved news feeds, overhauled navigation and theme design. CommunityPress also comes stock with events, user moderation, groups and forums. Try the Demo... CommunityPress.jessesoffice.com, PLEASE rate this plugin if you like it!
== Installation ==

You can download and install CommunityPress using the built in WordPress plugin installer. If you download CommunityPress manually, make sure it is uploaded to "/wp-content/plugins/communitypress/".

Activate CommunityPress in the "Plugins" admin panel using the "Activate" link.

You will need to enable permalink support in your WordPress installation for CommunityPress pages to function correctly. You can set this up using the "Settings > Permalinks" menu in your WordPress admin area.

Finally, you will need to activate the CommunityPress default theme. The theme is bundled with the plugin, you can activate these using the "Appearance > Themes" menu in your WordPress admin area.

== Frequently Asked Questions ==

= Can I use my existing WordPress theme? =

No. CommunityPress has a clean intuitive theme bundled with the plugin that is required for the plugin to work.

= Will this work on standard WordPress and WordPress MU? =

No. CommunityPress will run only on standard versions of WordPress. If you are using WordPress MU then check back for a future release with MU support.
